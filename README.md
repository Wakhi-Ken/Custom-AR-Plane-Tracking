# AR Tap-to-Place System (Unity AR Foundation)

A simple Augmented Reality placement system built with Unity AR Foundation. Users can tap on detected planes to spawn, switch, delete, scale, and rotate 3D objects in real time.

---

## Features

- Tap-to-place AR objects on detected planes  
- Switch between cube or sphere to spawn  
- Delete all spawned objects with one button  
- Pinch to scale objects  
- Two-finger rotate support  
- Runtime color control  
- Transparent / solid UI mode toggle  

---

## Controls

- Tap screen → Spawn selected object  
- Button 1 / Button 2 → Switch between cube or sphere  
- Delete button → Remove all spawned objects  
- Pinch (2 fingers) → Scale object  
- Twist (2 fingers) → Rotate object  
- UI toggle → Change transparency mode
